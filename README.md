# Verivo SoftwareThis is a repo for managing machine readable artifacts for the [Verivo Software APIs](http://www.verivo.com/). The goal is to maintain and evolve an OpenAPI, Postman Collection, and APIs.json index for the [Verivo Software APIs](http://www.verivo.com/).They are a constant work in progress, and will be updated on a regular basis when new information is added, or more of the specifications are supported.Feel free to submit a pull request to evolve any of the artifacts, and leave GitHub issues if you have any questions, comments, or feedback.- [Website](http://www.verivo.com/)- [Developer](http://www.verivo.com/)- [Twitter](https://twitter.com/verivosoftware)The [APIs.json](https://github.com/api-evangelist/verivo-software/blob/master/apis.json) index provides links to all the relevant links for the Verivo Software APIs, including the OpenAPI and Postman if they are present.
